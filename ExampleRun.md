# An Example Run to show all the functionality

## Dough Selection
NOTE: Every day you choose which dough(s) to prepare. You can get a buff to prepare multiple doughs
* You have infinite amount of this dough for the day. 
Options:
	- Laminated Dough
	- Sweet dough
	- Savory dough
	- Sourdough
	
## Starting cards
- Your selected dough influences the starting cards you get. 
- If you selected the Laminated Dough:
	Apologize
	Tell Joke
